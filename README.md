# Quiz Master Client
A simple client for quiz master app using ReactJS.

## Setup
- clone the repo

` git clone git@github.com:ryanermita/Quiz-Master-Client.git`

- install the project dependencies by running this command:

`yarn install`

- run the project using this command:

`yarn start`
